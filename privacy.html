Privacy Policy

Ultimo aggiornamento: 10/02/2026

Chi siamo

Questa applicazione (“App”) è utilizzata per automatizzare la creazione di didascalie (caption) per post social partendo da file presenti su Google Drive.

Dati che trattiamo

  • Metadati dei file (es. nome file) letti da Google Drive per generare una caption.
  • Informazioni dell’account social necessarie all’integrazione con Meta/Instagram (es. ID account) e token di accesso per pubblicare o preparare contenuti, quando autorizzato.
  • Contenuti generati (caption e testo) prodotti dall’automazione.

Come usiamo i dati

  • Per generare caption e testi collegati ai file selezionati.
  • Per inviare/gestire contenuti tramite le API Meta/Instagram quando l’utente autorizza l’accesso.

Condivisione dei dati

I dati possono essere condivisi con fornitori tecnici strettamente necessari al funzionamento dell’automazione, ad esempio:

  • Google (Drive) per l’accesso ai file autorizzati.
  • Meta (Instagram/Facebook) per l’integrazione e l’eventuale pubblicazione.
  • OpenAI per la generazione del testo (caption) a partire dai metadati forniti.

Conservazione

I dati vengono conservati solo per il tempo necessario all’esecuzione dell’automazione e per finalità operative (log tecnici), salvo obblighi di legge.

Le tue scelte

Puoi revocare in qualsiasi momento l’accesso a Google/Meta dalle impostazioni dei rispettivi account.

Eliminazione dei dati

Per richiedere la cancellazione dei dati legati alla tua integrazione, consulta le istruzioni qui: Data Deletion Instructions.
